Zero-Knowledge Mode

Enterprise customers can enable ZK mode where memory content never leaves your system. Sgraal validates using cryptographic proofs and receives only the omega score and proof hash. This mode is compliant with the most stringent data residency requirements.

Black Box Recorder

Every BLOCK decision generates a tamper-evident capsule with SHA256 hash chain. This provides cryptographic proof of what was blocked, when, and why — without revealing the memory content itself.
