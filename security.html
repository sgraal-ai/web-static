Zero-Knowledge Preflight

Memory content never leaves your infrastructure. SHA-256 proof hash returned instead of content.

POST /v1/preflight/zk

Use when: GDPR, HIPAA, data residency requirements.

Court Arbitration

Z3 SMT formal verification for multi-agent conflicts. No probabilistic system can override a formal verdict.

POST /v1/court/arbitrate
  • overridable: false
  • verdict_id + z3_proof in every response

Memory Passport

Cryptographically signed memory snapshots. Verifiable by any Sgraal instance.

POST /v1/memory/passport/export

3 TTL levels: ephemeral (5min), standard (1h), archival (30d)

Proof of Decision

Every preflight response includes cryptographic proof fields.

  • input_hash — SHA-256 of the input
  • proof_version — v1
  • deterministic: true — same input always produces same decision
  • reproducible: true — audit trail for every agent action
