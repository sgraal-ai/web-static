Beta · Early Access

For AI-driven trading desks, HFT, and quantitative funds

Determinism. Latency. Audit. No exceptions.

Trading agents need deterministic replay of every executed trade, sub-millisecond decision latency, and an audit trail that survives regulator review. The cost of a memory-poisoning incident in production is measured in basis points, not engineer-hours.

Edge mode and determinism ship today — both are production grade. Memory Time Machine for execution audit and compliance officer hooks for high-stakes trade interception are in Beta. Quant teams joining the design partner program get NDA-channel access.

Apply to design partner program → Try in playground Pricing →

Important — compliance evidence: /v1/check verdicts are not included in the audit trail and do not generate W3C Verifiable Credentials. For compliance-grade evidence (HIPAA, GDPR, EU AI Act, FDA 510(k), NIST AI RMF), use /v1/preflight, which produces full audit-log entries and signed W3C VCs per verdict. The /v1/check endpoint is suitable for experimentation and high-frequency agent gating only.

What works today

Five primitives the trading floor expects.

Production-grade, in active use, ready for your stack today.

bolt

Edge mode (sub-ms p99)

Live

Zero-dependency in-process scoring. Runs in the same process as the trading agent. Sub-millisecond p99 latency. No network call between the agent and the safety check. Air-gappable for isolated dealing rooms.

history_edu

Deterministic replay

Live

pg_override_disabled + cfg_checksum mode produces bit-perfect replay of any historical decision. Regulator audit, internal compliance review, post-incident root cause — all reconstructable.

call_merge

Subjective Logic multi-source fusion

Live

Jøsang opinion algebra for combining confidence signals from multiple data sources (market data, internal models, risk overlays). Byzantine-fault-tolerant aggregation when sources disagree.

badge

Signed memory passports

Live

Cryptographic attestation of memory provenance per tenant. The chain-of-custody for any decision is signed by your tenant key — never the vendor's.

trending_down

/v1/predict/degradation

Live

Returns the projected day a given memory state will cross the BLOCK threshold. Trading desks pre-schedule strategy refresh ahead of memory degradation — predictive, not reactive.

What is in active development

Two Beta features for trading desk design partners.

In active development with design partner input. Not yet GA — but real, not vaporware.

history

Memory Time Machine — execution audit

Beta

Deterministic replay of the full decision chain leading to a trade, with counterfactual branching. Useful for post-mortem on flagged trades, regulator queries, and internal model retrospectives.

Status: core replay works against test workloads. The compliance-export format is being shaped with design partners (regulator audit standard varies per jurisdiction).

pause_circle

In-Decision Human Veto (Schrödinger)

Beta

Pause an agent's execution mid-decision for human review on flagged high-stakes trades. Compliance officer hooks into the SSE decision stream and can veto before commit. Standard SSE protocol; the trading-desk-specific framing is the Beta part.

Status: SSE streaming is GA; the human-veto callback handshake is in test. Design partners get the first integration.
