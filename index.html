Python SDK

Integrate Sgraal directly into your Pythonic agent workflows.

import sgraal

sg = sgraal.Client("API_KEY")
ok, report = sg.evaluate(intent="update_schema")

Node.js SDK

Use Sgraal in your Node.js or TypeScript agent projects.

npm install @sgraal/sdk

const sgraal = require('@sgraal/sdk');
const sg = new sgraal.Client('API_KEY');
const { decision } = await sg.evaluate({
  intent: 'delete_user_data',
  context: 'Agent requested cleanup'
});
console.log(decision); // BLOCK
