Python SDK

Plain strings in, plain English out. One call.

import sgraal

sg = sgraal.Client("sg_live_...")
result = sg.check([
    "Deploy target is production",
    "API key is sk-proj-abc123"
])

print(result["safe"])    # False
print(result["reason"])  # "Memory contains a likely secret (API key)."

Node.js

Same simple interface. Plain strings in, plain English out.

const { SgraalClient } = require('@sgraal/mcp');
const sg = new SgraalClient('sg_live_...');

const result = await sg.check([
    "Customer approved for $50k credit line",
    "Last review was 18 months ago"
]);

console.log(result.safe);    // false
console.log(result.reason);  // "Stale data — review older than policy threshold."

REST API

Call /v1/check directly — no SDK required, any language.

curl -X POST https://api.sgraal.com/v1/check \
  -H "Authorization: Bearer sg_demo_playground" \
  -H "Content-Type: application/json" \
  -d '{"memories": ["Deploy target is production",
                  "API key is sk-proj-abc123"]}'

# {"safe": false, "reason": "Memory contains a likely secret (API key)."}
